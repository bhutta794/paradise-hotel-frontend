📋 Project Overview

Paradise Hotel is a complete hotel booking platform that allows users to search for available rooms, make reservations, and manage their bookings. The platform includes a comprehensive admin panel for managing locations, rooms, and viewing all reservations with analytics dashboards.


The application consumes a provided backend API (Express + SQLite) and presents a complete frontend experience using React with Material-UI components.

🚀 Installation & Setup

Prerequisites

  • Node.js (v18 or higher)
  • npm (comes with Node.js)

Step 1: Clone Repository

git clone https://github.com/bhutta794/paradise-hotel-frontend.git
cd paradise-hotel-frontend/paradise-hotel

Step 2: Install Dependencies

# Install frontend dependencies
npm install

# Install backend dependencies
cd "paradise hotel BE"
npm install
cd ..

Step 3: One-Command Installation (Alternative)

npm run install-all

Step 4: Run the Application

npm run dev

Access the Application

📡 API Endpoints

Public Endpoints

MethodEndpointDescription
POST/api/auth/registerCreate user account
POST/api/auth/loginLogin + JWT token
GET/api/locationsAll hotel locations
GET/api/rooms/availabilitySearch available rooms

Protected User Endpoints (JWT Required)

MethodEndpointDescription
GET/api/auth/meCurrent user info
POST/api/reservationsCreate reservation
GET/api/reservations/meUser's reservations
DELETE/api/reservations/:idCancel reservation

MethodEndpointDescription

🔧 Troubleshooting

IssueSolution
Backend won't startCheck if port 4000 is free: netstat -ano | findstr :4000
Frontend can't connectVerify backend is running at http://localhost:4000
Registration failsPassword must be 6+ characters, email must be unique
Images not showingUse direct image URLs (ending in .jpg, .png)
Port conflictsChange PORT in backend/.env and API_BASE_URL in frontend/src/services/api.js
